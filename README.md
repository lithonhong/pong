# Pong!


**Pong!** is a quick 2-player game where players control paddles to hit a ball in order to score goals at the opponent's side.

This project is created with the aim of familiarising myself with the Canvas API.

Play the game [here](https://lithonhong.github.io/pong/)!


## Further Implementation

As this project was intended to be a simple one-day implementation before moving forward to larger projects, only the essentials are included.

Possible improvements of this project:
* Mobile & accessibility support (touchscreen & buttons)
* Nuanced collision between the ball and paddles
* Audio controls
* Different modes, eg. player vs bot
